PDF Reader

[Usage] https://aws_deployment_address/YOUR_PDF_URL[?image=false&correct=false]

Query Parameters (optional)
  • image: whether to enable image extraction
  • correct: whether to enable text correction
Both of them default to be True, which means they default to be enabled.
If you want to disable any of them, you have to set the corresponding parameter to False.
The case of the bool value you set does not matter, which means either false or False will work.
